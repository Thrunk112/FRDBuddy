Commands:

"/frd" to toggle addon on or off. Off by default on log in

"/frd lock" to unlock or lock frame to allow drag to reposition

FRD Buddy creates a small weak-aura like icon that displays the durability of your Force Reactive Disk

If you talk to a vendor it will auto repair. If you leave a vendor window open, it will auto repair when the shield hits 25%
